Definition

$c$ : lightspeed

$m = m_e m^\ast$

$m^* = 1$

$q = q_e q^\ast$

$q^* = 1$

$v = cv^\ast$

$T$ ULF wave period

$x = c Tx^\ast$

$\vec{B} = B_{eq}\vec{B^\ast}$

$\vec{E} = B_{eq}\vec{E^\ast}$

$t = Tt^\ast$
