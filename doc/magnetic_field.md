# Magnetic_field

We use a dipole approximate magnetic field

$\vec{B}(\vec{m},\vec{r}) = \frac{\mu_0}{4\pi}\frac{3(\vec{m}\cdot \vec{\hat{r}})\vec{\hat{r}}-\vec{m}}{r^3}$

$\vec{\hat{r}}=\frac{\vec{r}}{r}$
